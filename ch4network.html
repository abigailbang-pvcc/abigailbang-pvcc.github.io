    Operating Systems Terminology

    Term

      Network Classifications

      • LAN - Local area network
      • MAN - Metropolitan area network
      • WAN - Wide area network

      Network Models

      • bus network - All machines are connected to a common comunication line called a bus.
      • star network - One machine serves as a central point where all others are connected.

      Protocols

      • CSMA/CD Ethernet - Carrier Sense, Multiple Access with Collision Detection. It dictates that each message be broadcast to all machines on the bus.
      • CSMA/CA wifi - Carrier Sense, Multiple Access with Collision Avoidance. Commonly referred to as wifi

      Combining Networks:

      • repeater - A device which exchanges signals between the two original buses
      • bridge - More complicated than a repeater, it connects buses, but does not pass all messages. It recognizes the destination addresses of messages, and form a connection when the message is intended for a computer on the other side.
      • switch - A bridge with multiple connections, which can connect multiple buses.
      • router - The devices which manage the connection between networks which form an internet.
      • gateway - The location that one network is joined to the internet.

      Client/Server Model

      • p2p: peer-to-peer - Client/server model which involves one process providing services to multiple clients.
      • distributed system - Software units that execute as processes on different computers.
      • cloud computing - Large numbers of shared computers on the network can be assigned for client use as needed.

      Internet Addressing

      • Tier 1 - High-speed, high-capacity, international WANs.
      • Tier 2 - Usually regional, and less capable than Tier 1.
      • Tier 3 - Also called an access. Provides access to the core of redundancy.

      IP Address

      • ICANN
      • DNS - Domain name system. They are an internet wide directory system of name servers.
      • TLD: Top-level domain - Examples are .edu, .gov, .org, .net, and .com.
      • Country-code TLDs

      Internet Protocols

      • FTP - File Transfer Protocol. It is an application to access a separate computer from a distance.
      • HTTP - Hyper Text Transfer Protocol. It handels web pages.
      • SMTP - Simple Main Transfer Protocol. A methold for town computers to interact on a network when transmitting an email.

      World Wide Web

      • URL - Uniform Resource Locator. It is a unique address given to web documents when retrieving and locating them.
      • Client-side activities - Activities performed by a client, such as in a browser.
      • server-side activities - Activities performed by a server, such as a webserver.

      Internet Four-Layer Software

      • application layer - One of the four layers of internet software, it is where a message typically originates.
      • transport layer - The seccond internet softwhere layer, where messages are transported to the network layer
      • network layer - The third layer of internet software, where the along with the transport layer the message is prepared for transmission.

      Internet Is:

      • packet-switching network -
      • fault tolerant
      • redundant

      Cybersecurity

      • malware - Malicious Software. It can be trasferred, and executed on the computer, or act from a distance.
      • virus - Software which inserts itself into existing programs withen the machine. Whe the "host" program is activated, the virus also attacks.
      • worm - Autonomous program that transfers itself through a network, residing in computers, and fowarding copies of itself to additional computers.
      • Trojan horse - A disguised program which seems useful or desirable, but is harmful.
      • spyware - Sometimes called Sniffing, it is a program that collects information about computer activities on the device where it is located, and reports it.
      • phishing - Unlike sniffing, which is secret, phishing consists of asking for information.
      • DDoS attack - Distributed denial of service. It is when the work of geerating attacks is distrbuted across multiple hosts.

      Protection

      • firewall - Can be used for individuals for protection, not networks or domains.
      • spam filter - A firewall variation, it blocks unwanted email content.
      • proxy server - It serves as an intermediary between a client and a server, shielding the client from averse server actions.
      • antivirus software - Detects and removes known viruses and infections.
      • cryptography - The study of securely sending and recieving messages in the presence of adversaries. It can use encryption to transform a message into a coded version.
      • public key encryption - Encryption and decryption algorithms are known, and one of the keys is known, but the encoded message is secure.
