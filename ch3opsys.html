    Operating Systems Terminology

    Term Definition
    Operating System Software which has control over the operation of the computer
    Job Queue It is the execution of jobs that reside in mass storage, withen batch processing systems.
    FIFO First-in, first-out is a way of retriving the oldest stored items first.
    Interactive Processing more
    Time Sharing more
    Multiprogramming It is a version of parallel programming. Multiple programs are run simlutaneously on a single processor. The OS executes parts of the different programs at slightly different times.
    Application software Programs which are used to complete tasks which relate to the machine's utilization.
    System software It performs common tasks for general computer systems.
    Utility software One of the categories of system software. It consists of mainly programs which perform important activites that are not included in the operating system.
    Graphical user interface (GUI) More modern option to replace shells. They are able to manipulate files and programs, and use icons on the display.
    Shell User interface which is old, and interacts with computer users through the use of textual messages.
    Kernel The internal part of an operating system, which contains software which performs basic functions.
    Device manager
    File manager It is a component of a kernel. It coordinates the mass storage on the machine, and records the files contained in mass storage.
    Memory manager A kernel component which coordinates the use of main memory.
    Virtual memory Created by paging, it is the fictional memory space.
    Boot strapping Performed by the computer when it is turned on, it moves the operating system from mass storage into main memory.
    ROM Read-only memory. It can be read, not altered.
